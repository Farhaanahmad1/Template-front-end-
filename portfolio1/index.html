Frequently asked questions

Can I use UI/UX Pro in open-source projects?

UI/UX Pro is an open-source library of interactive components built on top of UI/UX Pro including buttons, dropdowns, modals, navbars, and more.

Is there a Figma file available?

UI/UX Pro is first conceptualized and designed using the Figma software so everything you see in the library has a design equivalent in our Figma file.

What are the differences between UI/UX Pro and UI/UX?

The main difference is that the core components from UI/UX Pro are open source under the MIT license, whereas UI/UX Pro is a paid product.
